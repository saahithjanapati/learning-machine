# Learning Repository

This repo is organized so you can:
- `Learn`: read lesson summaries and concept notes by topic.
- `Practice`: fill in coding exercises and get targeted review.
- `Plan`: track your skill progression with a lightweight skill graph.

## Structure

- `topics/`
  - Topic-specific materials (notes, lessons, practice code).
- `learning_system/`
  - Cross-topic tracking files (lesson index, skill graph).

## Current Topic Coverage

- `transformers/kv-caching`

## Workflow (Recommended)

1. Pick a topic + mode (`learn` or `practice`).
2. Work in `topics/<domain>/<topic>/practice/` for exercises.
3. Save lesson recap in `topics/<domain>/<topic>/lessons/`.
4. Update `learning_system/LESSON_INDEX.md`.
5. Update `learning_system/SKILL_GRAPH.md` as skills improve.

Notation, Different Variants, etc
Many definitions, theorems have lots of different versions in the literature.
In HW questions, and Exams you have to use the version we used in class.
Examples:
* You can't say this:
"I understand that my solution is incorrect; however, if we were to use a different definition of the Lagrangian function, my solution would be valid"
* You can't say this either:
"In class this variable was defined to be positive, whereas I chose it to be negative (without saying so explicitly), following the convention used in some other texts"

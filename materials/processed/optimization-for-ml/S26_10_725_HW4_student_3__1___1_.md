**1 Testing Sum-of-Square via SDP [4 pts] (Julia)**

For certain optimization tasks, it is useful to decide if a polynomial $p(x)$ can be negative for any $x$; however, this may be a difficult task for multivariate polynomials, so we may aim for the next best thing: decide if $p(x)$ is a so-called “sum of square" polynomial. To make this problem more straight-forward, we only consider uni-variate polynomials with real coefficients.

Precisely, a degree $2d$ polynomial $p(x) = r_0 + r_1x + \dots + r_{2d}x^{2d}$ is a sum-of-square polynomial if and only if we can write $p(x)$ as
$$p(x) = \sum_{i=1}^m (q_i(x))^2$$
for some polynomials $q_1, \dots, q_m$ up to degree $d$. For example, $p(x) = x^2 + 2x + 3$ is a sum-of-square polynomial because we can write it as $p(x) = (x+1)^2 + (\sqrt{2})^2$. Evidently, sum-of-square polynomials are always non-negative.

Throughout the problem, we adopt the notation that $[x]_d = [1, x, x^2, \dots, x^d]^T$.

1.  Let $d$ be any positive integer, and $p(x) = \sum_{i=0}^{2d} r_i x^i$ be any polynomial of degree up to $2d$. Prove that $p(x)$ is sum-of-square if and only if there exists some symmetric positive semidefinite matrix $Q$ such that
    $$p(x) = [x]_d^T Q [x]_d.$$

**Solution** Write each $q_i$ in the monomial basis:
$$q_i(x) = a_i^T [x]_d$$
for some vector $a_i \in \mathbb{R}^{d+1}$.
If $p$ is SOS, then
$$p(x) = \sum_{i=1}^m (q_i(x))^2 = \sum_{i=1}^m ([x]_d^T a_i)^2 = \sum_{i=1}^m [x]_d^T a_i a_i^T [x]_d = [x]_d^T \left(\sum_{i=1}^m a_i a_i^T\right) [x]_d.$$
Define
$$Q := \sum_{i=1}^m a_i a_i^T.$$
Then $Q$ is symmetric and positive semidefinite, since it is a sum of rank-one PSD matrices. This proves the forward implication.

For the converse, suppose there is a symmetric PSD matrix $Q$ with
$$p(x) = [x]_d^T Q [x]_d.$$
Since $Q \ge 0$, we can factor it as $Q = B^T B$ (for example by Cholesky or the spectral theorem). Then
$$p(x) = [x]_d^T B^T B [x]_d = ||B[x]_d||^2.$$

$$\lambda^2 + (1 - \gamma) = 0.$$

5.1 Ridge Regression (3 Pts)

Ridge regression is a regularized version of linear regression which aims to combat overfitting. Given features $X \in \mathbb{R}^{n \times p}$, labels $y \in \mathbb{R}^n$, and weights $w \in \mathbb{R}^p$ ridge regression aims to minimize the following objective with regularization parameter $\lambda$

$\min ||y - Xw||^2 + \lambda||w||^2$

We can also think of this as a constrained minimization problem minimizing the ordinary squared loss but with an additional constraint on the weights $||w||^2 \le t$. Solve this problem with Lagrange multipliers for a closed form solution of ridge regression. **Solution** Consider the constrained problem

$\min ||y - Xw||^2 \quad \text{s.t.} \quad ||w||^2 \le t$.

Its Lagrangian is

$\mathcal{L}(\omega, \lambda) = ||y - X\omega||^2 + \lambda(||\omega||^2 - t), \quad \lambda \ge 0$.

Taking the gradient with respect to $\omega$ gives

$\nabla_\omega \mathcal{L} = -2X^T(y - X\omega) + 2\lambda\omega$.

Setting this to zero,

$-2X^T y + 2X^T X\omega + 2\lambda\omega = 0$,

so

$(X^T X + \lambda I)\omega = X^T y$.

$\omega(\lambda) = (X^T X + \lambda I)^{-1} X^T y$.

This is the ridge-regression solution. The KKT conditions imply the following:
* if the unconstrained least-squares solution already satisfies $||w||^2 \le t$, then $\lambda = 0$;
* otherwise $\lambda > 0$, and complementary slackness implies that the optimum lies on the boundary, i.e. $||w(\lambda)||^2 = t$.

Therefore, the closed-form ridge estimator is

$\omega_{ridge} = (X^T X + \lambda I)^{-1} X^T y$.

$p(x) = \frac{1}{\sqrt{2\pi\sigma}} \exp\left(-\frac{(x - \mu)^2}{2\sigma}\right)$.

Hence the maximum-entropy continuous distribution with fixed mean $\mu$ and variance $\sigma$ is the Gaussian $\mathcal{N}(\mu, \sigma)$.

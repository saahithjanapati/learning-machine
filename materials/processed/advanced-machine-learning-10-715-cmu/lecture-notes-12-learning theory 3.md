In the last lecture, we discussed the realizability and i.i.d. assumptions and introduced the fundamental theorem of statistical learning theory under those assumptions.

This lecture will cover shattering and VC dimension, which are independent of realizability and i.i.d assumptions. VC dimension is a combinatorial property of $\mathcal{H}$ entirely independent of any sample space.

### 10.1 VC dimension recap
The *restriction* of $\mathcal{H}$ onto $X$ $\mathcal{H}_X = \{ (h(X_1), \dots, h(X_m)) | h \in \mathcal{H} \}$ is the set of label vectors that can be generated by hypotheses in $\mathcal{H}$.

The *shattering number* $|\mathcal{H}_X|$ is the number of possible labelings of $X$, so $|\mathcal{H}_X| \le 2^{|X|}$.

$\mathcal{H}$ **shatters** $X$ if $|\mathcal{H}_X| = 2^{|X|}$ i.e. $\mathcal{H}$ can generate any possible labeling of $X$.

The **VC dimension** of $\mathcal{H}$ is the maximum size of a set that can be shattered by $\mathcal{H}$.

The *sample complexity* $m_{\mathcal{H}}(\epsilon, \delta)$ is the minimum number of training samples needed to find a probably approximately correct hypothesis. A lower bound on the number of samples needed for PAC learning is an upper bound on the sample complexity.

The *growth function* $\tau_{\mathcal{H}}(m) = \max_{|X|=m} |\mathcal{H}_X|$ measures how the shattering number grows with $|X|$.

### 10.2 Informal summary
Statistical learning is concerned with finding a needle ($h^*$) in a haystack ($\mathcal{H}$). Intuitively, the shattering number measures the effective size of our haystack on some $X$. If $\mathcal{H}$ can label $X$ in many different ways, then the search problem is difficult, and the sample complexity will be high.

In the extreme case, $\mathcal{H}$ shatters all points and any labeling is possible, which means that it is impossible to learn from a training subset how $h^*$ will behave on the rest of the points. The *no free lunch theorem* will formalize this notion, and we can show that if $|X| = \infty$ and $VC(\mathcal{H}) = \infty$, no learning is possible at all.

On the other hand, if the shattering number is small, then we only need to observe a few points to have a good idea of how $h^*$ will behave on all of $X$. It turns out that VC dimension completely characterizes the growth of the shattering number, and thus also characterizes sample complexity and PAC learnability (fundamental theorem of statistical learning theory). *Sauer's lemma* says that when $|X|$ is below the VC dimension, $\mathcal{H}$ shatters all points and the growth function is exponential; whereas above the VC dimension, the shattering number grows slowly (polynomially).

### 10.3 Sauer's Lemma
$$|\mathcal{H}_X| \le \sum_{i=0}^{VC(\mathcal{H})} \binom{|X|}{i}$$

For $m > VC(\mathcal{H})$, $|\mathcal{H}_X| \le \left( \frac{em}{VC(\mathcal{H})} \right)^{VC(\mathcal{H})}$ is polynomial in $m$, a remarkable result!

Reference: Shai page 74

*Proof.*
$$|\mathcal{H}_X| \le |\{B | B \subseteq X, \mathcal{H} \text{ shatters } B\}| \le \sum_{n=0}^{VC(\mathcal{H})} \binom{|X|}{n}$$

The second inequality adds up the number of ways to choose of set of size $n \le VC(\mathcal{H})$, since any larger set cannot be shattered by $\mathcal{H}$.

For the first inequality, we induct on $m = |X|$. (Base case, not covered in lecture: if $\mathcal{H}$ is empty, the inequality holds. For nonempty $\mathcal{H}$ and $m = 1$, either $|\mathcal{H}_X| = 1$ and $\mathcal{H}$ only shatters the empty set, or $|\mathcal{H}_X| = 2$ and $\mathcal{H}$ shatters the empty set and $X$.)

Induction: let $X' = \{X_2, \dots, X_m\}$. We list the unique elements of $\mathcal{H}_X$:
$$
\begin{matrix}
h_1(X_1), & \underbrace{h_1(X_2), \dots, h_1(X_m)} \\
h_2(X_1), & \underbrace{h_2(X_2), \dots, h_2(X_m)} \\
\vdots & \vdots \\
h_k(X_1), & \underbrace{h_k(X_2), \dots, h_k(X_m)}
\end{matrix}
$$
**Table 1.** Columns vary by $X$ and rows vary by hypothesis. Each entry is a 0 or 1 label, and rows are unique. By the induction hypothesis we have some idea about the bracketed columns.

The bracketed columns include all elements of $\mathcal{H}_{X'}$. In addition, some elements appear twice if $\mathcal{H}_X$ generates both $(0, y_2, \dots, y_m)$ and $(1, y_2, \dots, y_m)$. Let $\mathcal{H}'$ be the set of hypotheses corresponding to these rows, so $|\mathcal{H}_X| = |\mathcal{H}_{X'}| + |\mathcal{H}'_{X'}|$. By the induction hypothesis,
$$|\mathcal{H}_{X'}| \le |\{B | B \subseteq X', \mathcal{H} \text{ shatters } B\}|$$
$$|\mathcal{H}'_{X'}| \le |\{B | B \subseteq X', \mathcal{H}' \text{ shatters } B\}|$$
$$= |\{B \cup \{X_1\} | B \subseteq X', \mathcal{H}' \text{ shatters } B \cup \{X_1\}\}|$$
$$\le |\{B \cup \{X_1\} | B \subseteq X', \mathcal{H} \text{ shatters } B \cup \{X_1\}\}|$$

Finally, since subsets of $X$ either exclude or include $X_1$,
$$|\mathcal{H}_X| = |\mathcal{H}_{X'}| + |\mathcal{H}'_{X'}| \le |\{B | B \subseteq X, \mathcal{H} \text{ shatters } B\}| \quad \blacksquare$$

Scriber's note (not covered in lecture): Sauer's lemma can also be proved via *shifting*. We can change any "1" in Table 1 above to a "0" unless it would produce a row that is already in the table. We repeat this until no more entries can be shifted, producing a new table $\tilde{\mathcal{H}}_X$. All possible shifts exist in $\tilde{\mathcal{H}}_X$, so if a row contains columns with 1s, then those columns are shattered by $\tilde{\mathcal{H}}$. Thus $|\mathcal{H}_X| = |\tilde{\mathcal{H}}_X| \le \{B | B \subseteq X, \tilde{\mathcal{H}} \text{ shatters } B\}$. Finally, $VC(\tilde{\mathcal{H}}) \le VC(\mathcal{H})$ because if some columns were shattered after a shift then they were shattered before the shift.

### 10.4 No free lunch theorem
If the hypothesis space $\mathcal{H}$ is unconstrained, then for any learning algorithm given at most $\frac{1}{2}|X|$ training samples, the output $\hat{h}$ has significant risk:
$$P \left( R(\hat{h}) \ge \frac{1}{8} \right) \ge \frac{1}{7}$$
*Cor.* Any $\mathcal{H}$ with infinite VC dimension is not PAC learnable.

Reference: Shai page 61

*Proof.* Let $|X| = 2n$. Choose $h^* : X \to \{-1, 1\}$ uniformly at random from the $2^{2n}$ possibilities, and choose the test distribution $\mathcal{D}$ uniformly at random over $X$. The learning algorithm has at least a $\frac{1}{2}$ chance of being tested on an unseen point $U_i$, and in that case it can't do better than chance:*
$$R_{\mathcal{D}}(\hat{h}) \ge \frac{1}{2} R_U(\hat{h}) = \frac{1}{2} \cdot \frac{1}{2} = \frac{1}{4}$$
Finally, note that $R(\hat{h}) \le 1$, so $1 - R(\hat{h})$ is nonnegative, so by Markov's
$$P \left( R(\hat{h}) \le \frac{1}{8} \right) = P \left( 1 - R(\hat{h}) \ge \frac{7}{8} \right) \le \frac{\mathbb{E}[1-R(\hat{h})]}{7/8} \le \frac{3/4}{7/8} = \frac{6}{7} \implies P \left( R(\hat{h}) \ge \frac{1}{8} \right) \ge \frac{1}{7}. \quad \blacksquare$$

*Scriber's note: the proof in class was more formal:
$$\mathbb{E}_{h^*} \mathbb{E}_{x_1, \dots, x_n \sim \mathcal{D}} R_{\mathcal{D}, h^*}(h) = \mathbb{E}_{h^*} \mathbb{E}_{x_1, \dots, x_n \sim \mathcal{D}} P_{x \sim \mathcal{D}}[h(x) \neq h^*(x)]$$
$$= \mathbb{E}_{h^*} \mathbb{E}_{x_i} \frac{1}{2n} \sum_{j=1}^{2n} I(h(z_j) \neq h^*(z_j))$$
$$\ge \mathbb{E}_{h^*} \mathbb{E}_{i_1, \dots, i_n \sim \text{Unif}(1, 2n)} \frac{1}{2n} \sum_{j=1, i \notin \{i_1, \dots, i_n\}}^{2n} I(h(z_j) \neq h^*(z_j))$$
$$= \frac{1}{2n} \mathbb{E}_{i_1, \dots, i_n \sim \text{Unif}(1, 2n)} \sum_{j=1, i \notin \{i_1, \dots, i_n\}}^{2n} \mathbb{E}_{h^*} I(h(z_j) \neq h^*(z_j))$$
$$= \frac{1}{2n} \cdot n \cdot \frac{1}{2} = \frac{1}{4}$$

Let $X_1, X_2, \dots$ be a sequence of random variables such that
$$P\left(X_n = \frac{1}{n}\right) = 1 - \frac{1}{n^2} \text{ and } P(X_n = n) = \frac{1}{n^2}$$

(a) Does $X_n$ converge in probability?

$$\lim_{n \to \infty} P(|X_n - X| \ge \epsilon) = 0$$
Note that $P(X_n = \frac{1}{n}) + P(X_n = n) = 1$. So, $X_n \in \{\frac{1}{n}, n\}$ for all $n$.
I claim $X_n \xrightarrow{p} 0$.
$$P(|X_n - X| \ge \epsilon) = P(X_n \ge \epsilon)$$
Choose $n$ s.t. $n \ge \epsilon$ and also $\frac{1}{n} < \epsilon$
Then $\begin{cases} P(X_n = \frac{1}{n}) = 1 - \frac{1}{n^2} \\ P(X_n = n) = \frac{1}{n^2} \end{cases}$
So, $P(X_n = n) = P(X_n \ge \epsilon) = \frac{1}{n^2} \to 0$ as $n \to \infty$

So, $X_n$ converges in probability.

(b) Find $c$ so that the test has level $\alpha$.

$X_1 \sim \text{Exponential}(\theta/n)$
$Pr(X_1 > c) = \alpha$
$Pr(X_1 > c) = 1 - Pr(X_1 \leq c)$
$$= \int_c^\infty \frac{e^{-x/(\theta/n)}}{(\theta/n)} dx$$
$$= -\frac{n}{\theta} [e^{-x/(\theta/n)}]_c^\infty = \alpha$$
$$(-\frac{n}{\theta}) [0 - e^{-cn/\theta}] = \frac{n}{\theta} e^{-cn/\theta} = \alpha$$
$$\frac{\alpha \cdot \theta}{n} = e^{-c/\theta}$$
$$\ln\left(\frac{\alpha \cdot \theta}{n}\right) = -\frac{c}{\theta}$$
$$\boxed{\frac{-\ln\left(\frac{\alpha \cdot \theta}{n}\right)}{\theta} = c}$$

$$l(x_1, \dots, x_n \mid \beta) = \sum_{i=1}^n \left[\left(\frac{-x_i}{\beta t_i}\right) - \log(\beta t_i)\right]$$

Note, $l'' < 0$, so $\hat{\beta}$ is maximizer.

**Some potentially useful facts**

* $Var(X) = E(X^2) - (EX)^2, Cov(X, Y) = E(XY) - E(X)E(Y)$.
* For a collection of random variables, $Var(\sum_{i=1}^n a_i X_i) = \sum_{i=1}^n \sum_{j=1}^n a_i a_j Cov(X_i, X_j)$.
* The law of total expectation: $E[E[Y|X]] = E[Y]$
* The law of total variance: $Var(Y) = E(Var(Y|X)) + Var(E(Y|X))$.
* Convergence in probability: $X_n \xrightarrow{P} X$ if $\forall \epsilon > 0, \lim_{n \to \infty} P(|X_n - X| \ge \epsilon) = 0$.
* Convergence in distribution: $X_n \rightsquigarrow X$ if $\lim_{n \to \infty} F_n(t) = F(t)$, for all $t$ for which $F$ is continuous.
* If $Z \sim N(0, I)$ and $X = \mu + \Sigma^{1/2} Z$, then $X \sim N(\mu, \Sigma)$. Conversely, if $X \sim N(\mu, \Sigma)$, then $\Sigma^{-1/2}(X - \mu) \sim N(0, I)$.
* If $X \in \mathbb{R}^k$, and $a \in \mathbb{R}^k$ is a constant vector, then $E(a^T X) = a^T E(X)$ and $Var(a^T X) = a^T Var(X) a$. If $A$ is a constant matrix with $k$ columns, then $E(AX) = AE(X)$ and $Var(AX) = A Var(X) A^T$.
* Let $X \sim N(\mu, \Sigma)$. Then $a^T X \sim N(a^T \mu, a^T \Sigma a)$ and $(X - \mu)^T \Sigma^{-1} (X - \mu) \sim \chi_k^2$.
* $E(\hat{\theta}_n - \theta)^2 = \text{bias}^2(\hat{\theta}_n) + Var(\hat{\theta}_n)$

**Markov's Inequality** Let $X \ge 0$ and assume $\mu = E[X]$ exists. Then $\forall t > 0, P(X \ge t) \le \mu/t$.

**Chebyshev's Inequality** Assume $E[X] = \mu$ and $Var(X) = \sigma^2$ exist. Then $\forall t > 0, P(|X - \mu| \ge t\sigma) \le 1/t^2$.

**Hoeffding's inequality** Let $X_1, \dots, X_n$ be independent, $X_i \in [a_i, b_i]$, and $E[X_i] = 0$. Then
$$\forall t > 0, P(\bar{X} \ge t) \le \exp\left(-\frac{2n^2t^2}{\sum_{i=1}^n (b_i - a_i)^2}\right) \text{ and } P(|\bar{X}| \ge t) \le 2 \exp\left(-\frac{2n^2t^2}{\sum_{i=1}^n (b_i - a_i)^2}\right).$$

**Delta method** Let $Y_n = (Y_{n1}, \dots, Y_{nk})^T$ be a sequence of random vectors, or random variables if $k=1$, and $g: \mathbb{R}^k \to \mathbb{R}$ be a differentiable function with derivative
$$\nabla g(y) = \left( \frac{\partial g}{\partial y_1} \frac{\partial g}{\partial y_2} \dots \frac{\partial g}{\partial y_k} \right)^T.$$
If $\sqrt{n}(Y_n - \mu) \rightsquigarrow N(0, \Sigma)$, then $\sqrt{n}(g(Y_n) - g(\mu)) \rightsquigarrow N(0, \nabla g(\mu)^T \Sigma \nabla g(\mu))$, where $\nabla g(\mu)$ is $\nabla g(y)$ evaluated at $y = \mu$.

**Fisher Information** Define the log-likelihood function $\ell_n(\theta) = \sum_{i=1}^n \log f_\theta(X_i)$. The score function
$$s_n(X; \theta) = \nabla_\theta \ell_n(\theta) = \sum_i \nabla_\theta \log f_\theta(X_i) = \sum_i s_1(X_i; \theta)$$
The Fisher information, $I_n(\theta) = Var_\theta(s_n(X; \theta)) = \sum_i Var_\theta(s_1(X_i; \theta)) = n I_1(\theta)$, where
$$I_1(\theta) = Var_{X \sim f_\theta}(s_1(X; \theta)) = -E_{X \sim f_\theta} \nabla_\theta^2 \ell_1(\theta).$$

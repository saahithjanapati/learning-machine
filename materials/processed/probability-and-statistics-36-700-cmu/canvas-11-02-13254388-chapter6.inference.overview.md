## 1 Statistical models

A central concern of statistics and machine learning is to understand/estimate things about some underlying population on the basis of samples. Formally, given a sample,
$$X_1, \dots, X_n \sim F,$$
what can we infer about $F$? There are three broad types of inferences: point estimation, interval estimation, and test of hypotheses. We will cover these topics in more details later on but we give is a quick preview in the next sections.

But first, to make meaningful inferences about $F$ from samples we typically restrict $F$ in some natural way. For example, if the process I am considering is continuous, I will not consider discrete models. We will denote by $\mathcal{F}$ the set of possible distributions $F$. This is called the **statistical model**. Broadly, there are two possibilities:

1. **Parametric model:** In a parametric model, the set of possible distributions $\mathcal{F}$ can be described by a finite number of parameters. Here are a few examples:
    (a) A Gaussian model: This is a simple two-parameter model. Here we suppose that:
    $$\mathcal{F} = \left\{ f(x; \mu, \sigma) = \frac{1}{\sigma\sqrt{2\pi}} \exp \left\{ -\frac{(x-\mu)^2}{2\sigma^2} \right\}, \mu \in \mathbb{R}, \sigma > 0, x \in \mathbb{R} \right\}.$$
    (b) The Bernoulli model: This is a one-parameter model where:
    $$\mathcal{F} = \left\{ \mathbb{P}_p(X = x) = p^x(1 - p)^{1-x} : 0 \le p \le 1, x = \{0, 1\} \right\}.$$

If we know/learn from data the value of the parameters, then we know the distribution completely.

2. **Non-parametric model:** A non-parametric model is one which where $\mathcal{F}$ cannot be parameterized by a finite number of parameters. Here are a few popular examples:
    (a) CDF estimation: Here the model consists of any valid CDF, i.e. a function that is between 0 and 1, is monotonically increasing, right-continuous and equal to 0 at $-\infty$ and 1 at $\infty$. We are given samples $X_1, \dots, X_n \sim F$ and the goal is to estimate $F$.
    (b) PDF/PMF estimation: In density estimation, we are given samples $X_1, \dots, X_n \sim f_X$, where $f_X$ is an unknown density that we would like to estimate. It turns out that the class of all possible densities is too big for this problem to be well posed so we need to assume some smoothness on the density. A typical assumption is that the model is given by:
    $$\mathcal{F} = \left\{ f : \int (f''(x))^2 dx < \infty, \int f(x)dx = 1, f(x) \ge 0 \right\}.$$

## 2 Point Estimation

Point estimation in statistics refers to calculating a single "best guess" of the value of an unknown quantity of interest $\theta$. The quantity of interest could be a parameter ($\mu, \sigma^2$ or $p$ in the parametric models above) or any other function of the data, for instance, a density function, $\theta = f$, or the interquartile range of a random variable, $\theta = F^{-1}(0.75) - F^{-1}(0.25)$.

Typically, we will use $\hat{\theta}$ or $\hat{\theta}_n$ to denote a point estimator. A point estimator is a function of the data $X_1, \dots, X_n$:
$$\hat{\theta}_n = g(X_1, \dots, X_n),$$
for example $\hat{\theta} = \bar{X}_n$, so that $\hat{\theta}_n$ is a random variable. We like estimators that have good properties, for example no or low bias, where bias is:
$$b(\hat{\theta}_n) = \mathbb{E}_\theta(\hat{\theta}_n) - \theta,$$
small variance
$$v(\hat{\theta}_n) = \mathbb{E}_\theta(\hat{\theta}_n - \mathbb{E}_\theta(\hat{\theta}_n))^2,$$
consistency, i.e. when the estimator $\hat{\theta}_n$ converges in probability to $\theta$.

**Remark:** $\theta$ is typically a function of $F$, i.e., $\theta = \theta(F)$; for example, if $\theta$ is the mean of a distribution, then $\theta(F) = \int x dF(x)$, or in the examples above, $\theta(F) = F^{-1}(0.75) - F^{-1}(0.25)$

and $\theta(F) = f = dF$. The $\mathbb{E}(\cdot)$ and $\mathbb{P}(\cdot)$ used below refer to the randomness in the data $X_1, \dots, X_n$ iid from $F$. Sometimes we use $\mathbb{E}_\theta(\cdot)$ and $\mathbb{P}_\theta(\cdot)$, like we did above, to emphasize that the underlying distribution is the one associated with $\theta$.

**The Bias-Variance decomposition** One way to compute the quality of an estimator is via its mean squared error:
$$MSE = \mathbb{E}_\theta(\theta - \hat{\theta}_n)^2.$$
The MSE can be decomposed as the sum of the squared bias and variance, i.e.:
$$MSE = \mathbb{E}_\theta[\theta - \hat{\theta}_n]^2$$
$$= \mathbb{E}_\theta[\theta - \mathbb{E}_\theta(\hat{\theta}_n) + \mathbb{E}_\theta(\hat{\theta}_n) - \hat{\theta}_n]^2$$
$$= [b(\hat{\theta}_n)]^2 + v(\hat{\theta}_n).$$

In classical statistics, often the starting point was to identify *unbiased* estimators, and then find unbiased estimators with small (or minimal variance). In modern statistics, we often aim to minimize the mean squared error of a biased estimator, because the reduction in variance often justifies the bias. Indeed, the MSE of an unbiased estimator can be larger than the MSE of a biased estimator, so unbiasedness is not necessarily the best property for an estimator.

**Consistency** We call an estimator $\hat{\theta}_n$ of a parameter $\theta$ *consistent* if $\hat{\theta}_n \xrightarrow{P} \theta$ (equivalently $\hat{\theta}_n - \theta = o_P(1)$), i.e. for any $\epsilon$:
$$\mathbb{P}(|\hat{\theta}_n - \theta| \ge \epsilon) \to 0,$$
as $n \to \infty$.

Note that a simple consequence of the bias-variance decomposition above is that: if $b(\hat{\theta}_n) \to 0$, i.e. the estimator is asymptotically unbiased, and $v(\hat{\theta}_n) \to 0$ then the estimator $\hat{\theta}_n$ is consistent since convergence in quadratic mean implies convergence in probability.

**Example:** Suppose $X_1, \dots, X_n \sim \text{Ber}(p)$, and our estimator is:
$$\hat{p}_n = \frac{1}{n} \sum_{i=1}^n X_i.$$
What is the bias of this estimator? What is its variance? Is the estimator consistent?

**Asymptotic Normality** Often estimators that we study will have an asymptotically normal distribution. This means that:
$$\frac{\hat{\theta}_n - \theta}{\sqrt{v(\hat{\theta}_n)}}$$
converges in distribution to $N(0, 1)$. We will refer to this property as asymptotic normality.

## 3 Confidence Sets

In general, for a parameter $\theta$ we define a $1 - \alpha$ confidence set $C_n$ to be any random set which has the property that:
$$\mathbb{P}_\theta(\theta \in C_n) \ge 1 - \alpha$$
for all $\theta$. We refer to $\mathbb{P}_\theta(\theta \in C_n)$ as the coverage of the confidence set $C_n$. The confidence set $C_n$ is a random set (and $\theta$ is a fixed parameter).

One can think about the coverage guarantee in the following way:

You repeat the experiment many times, each time constructing a different confidence interval $C_n$. Then $1 - \alpha$ of these different sets will contain the corresponding true parameter. Notice, that the true parameter does not have to be fixed, so in some sense the experiment you conduct can be different each time.

**Example:** We already saw a way to construct confidence intervals for a Bernoulli parameter using Hoeffding's inequality, which we state again here:

Suppose that $X_1, \dots, X_n$ are independent and that, $a_i \le X_i \le b_i$, and $\mathbb{E}[X_i] = 0$. Then for any $t > 0$:
$$\mathbb{P}(|\bar{X}| \ge t) \le 2 \exp \left( -\frac{2n^2t^2}{\sum_{i=1}^n (b_i - a_i)^2} \right).$$

Now let $\hat{p} = n^{-1} \sum_i X_i$ be the fraction of tosses that are heads; it has mean $p$, so consider the random variable $\hat{p} - p$, which has mean 0, and is bounded between $-p$ and $1 - p$, so that $(b_i - a_i) = 1$. Fix $\alpha > 0$ and let,
$$t = \sqrt{\frac{1}{2n} \log(2/\alpha)} > 0.$$
By Hoeffding's inequality,
$$\mathbb{P}(|\hat{p} - p| \ge t) \le 2 \exp(-2nt^2) = \alpha,$$

or equivalently
$$\mathbb{P}(|\hat{p} - p| < t) = \mathbb{P}(\hat{p} - t \le p \le \hat{p} + t) > 1 - \alpha,$$
Let
$$C_n = \left( \hat{p}_n - \sqrt{\frac{\log(2/\alpha)}{2n}}, \hat{p}_n + \sqrt{\frac{\log(2/\alpha)}{2n}} \right).$$
The random interval $C_n$ traps the true parameter value $p$ with probability at least $1 - \alpha$; we call $C_n$ a $1 - \alpha$ confidence interval.

More generally, we can always use concentration inequalities to construct confidence intervals. These confidence intervals are often loose so we often (or almost always) use instead approximate (asymptotic) confidence intervals.

Indeed it is often the case that:
$$Z = \frac{\hat{\theta}_n - \theta}{\sqrt{v(\hat{\theta}_n)}} \to N(0, 1), \quad n \to \infty,$$
in which case $\hat{\theta}_n \dot{\sim} N(\theta, v(\hat{\theta}_n))$. Define, $z_{\alpha/2} = \Phi^{-1}(1 - \alpha/2)$. Then we would construct a confidence interval:
$$C_n = \left( \hat{\theta}_n - z_{\alpha/2}\sqrt{v(\hat{\theta}_n)}, \hat{\theta}_n + z_{\alpha/2}\sqrt{v(\hat{\theta}_n)} \right).$$

We now need to verify that:
$$\mathbb{P}_\theta(\theta \in C_n) \to 1 - \alpha,$$
as $n \to \infty$, which what it means to be an asymptotic confidence interval.
$$\mathbb{P}_\theta(\theta \in C_n) = \mathbb{P} \left( \hat{\theta}_n - z_{\alpha/2}\sqrt{v(\hat{\theta}_n)} \le \theta \le \hat{\theta}_n + z_{\alpha/2}\sqrt{v(\hat{\theta}_n)} \right)$$
$$= \mathbb{P}_\theta \left( -z_{\alpha/2} \le \frac{\hat{\theta}_n - \theta}{\sqrt{v(\hat{\theta}_n)}} \le z_{\alpha/2} \right)$$
$$\to \mathbb{P}(-z_{\alpha/2} \le Z \le z_{\alpha/2}) = 1 - \alpha.$$

**Example:** We previously constructed Bernoulli confidence sets using Hoeffding's inequality. Here we use a normal approximation since the estimator $\hat{p}$ is an average, to which the CLT applies. But first we note that the variance of our estimator is:
$$v(\hat{\theta}_n) = \frac{p(1 - p)}{n}$$

## 4 Hypothesis testing

Typically, the way that statistical hypothesis testing proceeds is by defining a so-called **null hypothesis**. We then collect data, and typically the question we ask is whether the data provides enough evidence to *reject* the null hypothesis.

**Example:** Suppose $X_1, \dots, X_n \sim \text{Ber}(p)$, and we want to test if the coin is fair. In this case the null hypothesis would be:
$$H_0 : p = 1/2.$$
We typically also specify a **alternate hypothesis**. In this case, the alternative hypothesis is:
$$H_1 : p \neq 1/2.$$
Typically, hypothesis testing proceeds by defining a **test statistic**. In this case, a natural statistic might be:
$$T = \left| \frac{1}{n} \sum_{i=1}^n X_i - p \right|.$$
It might make sense to reject the null hypothesis if $T$ is large. We will be more precise about this later on particularly by defining the different types of errors, and how to set the threshold for $T$.

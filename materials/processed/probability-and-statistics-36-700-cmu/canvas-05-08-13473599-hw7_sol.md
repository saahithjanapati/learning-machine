1. Wasserman Exercise 6 on page 147
**Solution:**
In this problem, we denote $\hat{\theta}$ to be the MLE for $\theta$ given $X_1, \dots, X_n \sim N(\theta, 1)$. We already know $\hat{\theta} = \overline{X}_n$ and that $\hat{se} = 1/\sqrt{n}$ (note that the variance is known). Further, we will write $\Phi$ as the standard normal CDF and $\phi = \Phi'$ as the standard normal PDF. We write $Z = X_1 - \theta$ and note $Z \sim N(0, 1)$.

(a):
$$\psi = \mathbb{P}(Y_1 = 1) = \mathbb{P}(X_1 > 0) = \mathbb{P}(X_1 - \theta > -\theta)$$
$$= \mathbb{P}(Z > -\theta) = 1 - \mathbb{P}(Z < -\theta) = 1 - \Phi(-\theta).$$
Here we used that $Z \sim N(0, 1)$. By equivariance of the mle, the mle $\hat{\psi}$ is $1 - \Phi(-\hat{\theta}) = 1 - \Phi(-\overline{X}_n)$.

(b): By symmetry of the CDF, one can check that
$$1 - \Phi(-\theta) = \mathbb{P}(Z > -\theta) = \mathbb{P}(Z < \theta) = \Phi(\theta).$$
Then we can rewrite $\psi = g(\theta)$ where $g(\theta) := \Phi(\theta)$. Then, $g'(\theta) = \phi(\theta)$. We thus compute the standard error of $\hat{\psi}$ as
$$\hat{se}(\hat{\psi}) = \hat{se}(\hat{\theta})|g'(\hat{\theta})| = \frac{\phi(\hat{\theta})}{\sqrt{n}} = \frac{\phi(\overline{X}_n)}{\sqrt{n}}.$$
An approximate 95 per cent confidence interval for $\psi$ is
$$\underbrace{1 - \Phi(-\overline{X}_n)}_{=\hat{\psi}} \pm 1.96 \cdot \frac{\phi(\overline{X}_n)}{\sqrt{n}}.$$

(c): By linearity of expectation, $\tilde{\psi}$ has mean $\mathbb{E}(Y_1) = \mathbb{P}(Y_1 = 1) = \psi$. Consistency follows from the weak law of large numbers.

(d): Note that $Y_1 \sim \text{Bernoulli}(\psi)$ so $\text{Var}(Y_1) = \psi(1 - \psi)$ and $\text{Var}(\tilde{\psi}) = \text{Var}(Y_1)/n = \psi(1 - \psi)/n$. The asymptotic relative efficiency (ARE) is
$$\frac{\phi^2(\theta)}{\psi(1 - \psi)} = \frac{\phi^2(\theta)}{\Phi(\theta)(1 - \Phi(\theta))}.$$

(e): Suppose the $X_i$ are drawn from some non-normal distribution with mean $\mu$. By the law of large numbers, $\overline{X}$ converges in probability to $\mathbb{E}(X_1) \equiv \mu$. So $\hat{\psi} = 1 - \Phi(-\overline{X})$ converges in probability to $1 - \Phi(-\mu) = \Phi(\mu)$. The true value of $\psi$ is $\mathbb{P}(X > 0) = 1 - \mathbb{P}(X < 0) = 1 - F_X(0)$. For an arbitrary distribution $F_X$, we have $1 - F_X(0) \neq \Phi(\mu)$ so the mle is inconsistent. On the other hand, $\tilde{\psi}$ is still consistent.

2. Let $X_1, \dots, X_n \sim N(\mu, \sigma^2)$. We will define a two-dimensional parameter $\theta := (\theta_1, \theta_2) := (\mu, \sigma^2)$.

(a) Derive the (two-dimensional) MLE $\hat{\theta}$.
**Solution:** The log likelihood of $\theta$ is
$$\ell(\theta; X_1, \dots, X_n) = -\frac{n}{2} \log(2\pi\sigma^2) - \frac{1}{2\sigma^2} \sum_{i=1}^n (X_i - \mu)^2.$$
Taking derivative with respect to $\theta$ yields
$$\frac{\partial}{\partial \mu} \ell(\theta; X_1, \dots, X_n) = \frac{1}{2\sigma^2} \sum_{i=1}^n 2(X_i - \mu)$$
$$\frac{\partial}{\partial \sigma^2} \ell(\theta; X_1, \dots, X_n) = -\frac{n}{2\sigma^2} + \frac{1}{2(\sigma^2)^2} \sum_{i=1}^n (X_i - \mu)^2 \tag{1}$$
With the first order optimality condition, the MLE of $\theta$ is
$$\hat{\mu} = \overline{X}_n, \quad \hat{\sigma}^2 = \frac{1}{n} \sum_{i=1}^n (X_i - \overline{X}_n)^2.$$

(b) The score function $s(\theta)$ is now a two-dimensional vector valued function (each coordinate is partial derivative of the log-likelihood with respect to each of the parameters). The Fisher information is now a $2 \times 2$ matrix,
$$I(\theta) = \mathbb{E}[s(\theta)s(\theta)^T].$$
Find $I(\theta)$ in the above setting. Make sure to simplify as much as possible.
**Solution:** The score function is, from (1),
$$s(\theta) = \left[ \frac{X - \mu}{\sigma^2}, -\frac{1}{2\sigma^2} + \frac{(X - \mu)^2}{2(\sigma^2)^2} \right]^\top.$$
Let $I(\theta)_{ij} = [I(\theta)]_{ij}$.
$$I(\theta)_{11} = \mathbb{E} \left[ \frac{(X - \mu)^2}{\sigma^4} \right] = \frac{1}{\sigma^2},$$
$$I(\theta)_{21} = \mathbb{E} \left[ \frac{(X - \mu)}{\sigma^2} \left( -\frac{1}{2\sigma^2} + \frac{(X - \mu)^2}{2(\sigma^2)^2} \right) \right]$$
$$= \frac{1}{2\sigma^4} \mathbb{E} \left[ (X - \mu) \left( -1 + \frac{(X - \mu)^2}{\sigma^2} \right) \right]$$
$$= \frac{1}{2\sigma^4} \mathbb{E} \left[ -(X - \mu) + \frac{(X - \mu)^3}{\sigma^2} \right]$$
$$= \frac{1}{2\sigma^6} \mathbb{E}(X - \mu)^3 = 0,$$
$$I(\theta)_{22} = \frac{1}{4\sigma^4} \mathbb{E} \left[ -1 + \frac{(X - \mu)^2}{\sigma^2} \right]^2$$
$$= \frac{1}{4\sigma^4} \mathbb{E} \left[ 1 - 2 \frac{(X - \mu)^2}{\sigma^2} + \frac{(X - \mu)^4}{\sigma^4} \right] = \frac{1}{2\sigma^4}.$$

(c) In this multivariate setting, $\sqrt{n}(\hat{\theta} - \theta) \to N(0, I(\theta)^{-1})$, in distribution. Use this to construct an asymptotically valid $1 - \alpha$ confidence interval for $\theta_2$.
**Solution:** Since we are only interested in $\theta_2 = \sigma^2$, by Slutsky’s Theorem,
$$\sqrt{n}(\hat{\sigma}^2 - \sigma^2) \rightsquigarrow N(0, 2\sigma^4).$$
[Alternatively, apply Wasserman Theorem 9.27 (Eq 9.19).] This implies that
$$\mathbb{P} \left[ |\hat{\sigma}^2 - \sigma^2| \le z_{\alpha/2} \frac{\sqrt{2}\sigma^2}{\sqrt{n}} \right] \to 1 - \alpha.$$
Therefore, an asymptotically valid $1 - \alpha$ confidence interval for $\sigma^2$ is
$$\hat{\sigma}^2 \left[ 1 - z_{\alpha/2} \frac{\sqrt{2}}{\sqrt{n}}, 1 + z_{\alpha/2} \frac{\sqrt{2}}{\sqrt{n}} \right].$$
*Remark. This is called Wald confidence set.*

(a) If a random variable $Y$ has distribution $P$, prove that the median $m$ of $P$ is the solution to $\min_{c \in \mathbb{R}} \mathbb{E}|Y - c|$. *Hint: If $Z$ is a non-negative random variable, $\mathbb{E}[Z] = \int_0^\infty \mathbb{P}(Z \ge t)dt$. Use this fact to rewrite $\mathbb{E}|Y - c|$ as the sum of two integrals, one from $c$ to $\infty$, one from $-\infty$ to $c$. Then apply the fundamental theorem of calculus and optimize in $c$. Argue that the optimal $c$ is the median.*
**Solution:**
Since $|Z - c|$ is a non-negative random variable, we write
$$\mathbb{E}|X - c| = \int_0^\infty \mathbb{P}(|X - c| \ge t)dt$$
$$= \int_0^\infty \mathbb{P}(X - c \ge t)dt + \int_0^\infty \mathbb{P}(X - c \le -t)dt$$
$$= \int_c^\infty \mathbb{P}(X \ge u)du + \int_{-\infty}^c \mathbb{P}(X \le u)dt.$$
In the second line, we noted $\{X - c \ge t\}$ is the disjoint union of the events $\{X - c \ge t\}$ and $\{X - c \le -t\}$. In the third line, we used a substitution $u = t + c$ and $u = -t + c$ in the respective integrals.
We want to optimize in the variable $c$, so differentiating in $c$ (relying on the Fundamental Theorem of Calculus and chain rule—check the details!), we obtain
$$\frac{d}{dc} \mathbb{E}|X - c| = \mathbb{P}(X \le c) - \mathbb{P}(X \ge c) = 2\mathbb{P}(X \le c) - 1.$$

(b) Given a prior $f(\theta)$, posterior density $f(\theta|x)$, loss function $L(\theta, \hat{\theta})$, recall that the Bayes estimator $\hat{\theta}(x)$ minimizes the posterior risk $r(\hat{\theta}|x) = \int L(\theta, \hat{\theta}(x))f(\theta|x)d\theta$. Use part (a) to prove that if you use the absolute error loss $L(\theta, \hat{\theta}) = |\theta - \hat{\theta}|$, the Bayes estimator is the median of the posterior distribution with density $f(\theta|x)$. *Hint: Theorem 12.8 of Wasserman gives a similar proof with the squared error loss.*
**Solution:** We must minimize $r(\hat{\theta}|x) = \int |\theta - \hat{\theta}(x)|f(\theta|x)d\theta$ in $\hat{\theta}(x)$. Well
$$\min_{\hat{\theta}(x)} r(\hat{\theta}|x) = \min_{\hat{\theta}(x)} \int |\theta - \hat{\theta}(x)|f(\theta|x)d\theta = \min_{\hat{\theta}(x)} \mathbb{E}_{\theta \sim f(\theta|x)} |\theta - \hat{\theta}(x)|.$$
We therefore apply part (a) where $P$ is the posterior distribution and $Y$ is the random variable $\theta$ drawn from $P$. The Bayes estimator is thus the median of $P$, the posterior density.

6. **[Delta method vs non-parametric and parametric bootstrap]**
Wasserman Exercise 9 pp.147-148

**Solution:** (a) The sample mean $\overline{X}$ is the MLE for $\mu$ and satisfies $\overline{X} \approx N(\mu, 1/n)$. We define $g(\mu) = e^\mu$ so that $\theta = g(\mu)$ and $\hat{\theta} = g(\overline{X}) = e^{\overline{X}}$. By the delta method,
$$SE_\Delta(\hat{\theta}) \approx \sqrt{\frac{1}{n} (g'(\overline{X}))^2} = \sqrt{\frac{1}{n} (e^{\overline{X}})^2} = \frac{e^{\overline{X}}}{\sqrt{n}} = \frac{\hat{\theta}}{\sqrt{n}}.$$
A Wald-type 95% confidence interval for $\theta$ is then
$$\hat{\theta} \pm 1.96 \cdot SE_\Delta(\hat{\theta}).$$
(a) Delta method: $\hat{SE} = 14.76323$, 95% CI = (118.6964, 176.5683)
(b) Parametric bootstrap: $\hat{SE} = 14.96014$, 95% CI = (121.0013, 179.3386)
(c) Nonparametric bootstrap: $\hat{SE} = 15.0753$, 95% CI = (118.0847, 177.1799)
All of the estimates are fairly close to each other.

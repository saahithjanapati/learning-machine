1. **[Power of the Wald test]**
(a) Prove Theorem 10.6 in Wasserman page 153.
**Solution:** We reject the null hypothesis $H_0 : \theta = \theta_0$ in favor of $H_1 : \theta \neq \theta_0$ if the absolute value of the Wald statistic exceeds the critical value:
$$\left| \frac{\hat{\theta} - \theta_0}{\widehat{\text{se}}} \right| > z_{\alpha/2}.$$
The power $\beta(\theta_\star)$ is the probability of this event occurring when the true parameter is $\theta_\star$. We can write this as the sum of the probabilities of the two rejection regions:
$$\beta(\theta_\star) = P_{\theta_\star} \left( \frac{\hat{\theta} - \theta_0}{\widehat{\text{se}}} < -z_{\alpha/2} \right) + P_{\theta_\star} \left( \frac{\hat{\theta} - \theta_0}{\widehat{\text{se}}} > z_{\alpha/2} \right).$$
Since the true value is $\theta_\star$, we know that the random variable $Z = \frac{\hat{\theta} - \theta_\star}{\widehat{\text{se}}}$ follows an approximate standard normal distribution, $Z \sim N(0, 1)$. We rearrange the inequalities above to express them in terms of $Z$.

**Term 1 (Left Tail):**
$$P_{\theta_\star} \left( \frac{\hat{\theta} - \theta_0}{\widehat{\text{se}}} < -z_{\alpha/2} \right) = P_{\theta_\star} \left( \hat{\theta} < \theta_0 - z_{\alpha/2} \widehat{\text{se}} \right)$$
$$= P_{\theta_\star} \left( \frac{\hat{\theta} - \theta_\star}{\widehat{\text{se}}} < \frac{\theta_0 - \theta_\star - z_{\alpha/2} \widehat{\text{se}}}{\widehat{\text{se}}} \right)$$
$$= P \left( Z < \frac{\theta_0 - \theta_\star}{\widehat{\text{se}}} - z_{\alpha/2} \right)$$
$$= \Phi \left( \frac{\theta_0 - \theta_\star}{\widehat{\text{se}}} - z_{\alpha/2} \right).$$

**Term 2 (Right Tail):**
$$P_{\theta_\star} \left( \frac{\hat{\theta} - \theta_0}{\widehat{\text{se}}} > z_{\alpha/2} \right) = P_{\theta_\star} \left( \hat{\theta} > \theta_0 + z_{\alpha/2} \widehat{\text{se}} \right)$$
$$= P_{\theta_\star} \left( \frac{\hat{\theta} - \theta_\star}{\widehat{\text{se}}} > \frac{\theta_0 - \theta_\star + z_{\alpha/2} \widehat{\text{se}}}{\widehat{\text{se}}} \right)$$
$$= P \left( Z > \frac{\theta_0 - \theta_\star}{\widehat{\text{se}}} + z_{\alpha/2} \right)$$
$$= 1 - \Phi \left( \frac{\theta_0 - \theta_\star}{\widehat{\text{se}}} + z_{\alpha/2} \right).$$

Combining these two terms yields the result:
$$\beta(\theta_\star) \approx 1 - \Phi \left( \frac{\theta_0 - \theta_\star}{\widehat{\text{se}}} + z_{\alpha/2} \right) + \Phi \left( \frac{\theta_0 - \theta_\star}{\widehat{\text{se}}} - z_{\alpha/2} \right).$$

(b) See Remark 10.5 on an alternative version of the Wald test statistic, where we in the denominator replace $\widehat{\text{se}}$ with the standard error of $\hat{\theta}$ computed at $\theta = \theta_0$. Is the power function of this version of the Wald test the same as in Part (a)? If not, what changes?
**Solution:** No, the power function is not exactly the same.
Let the alternative Wald statistic be defined as $W = \frac{\hat{\theta} - \theta_0}{\text{se}_0}$, where $\text{se}_0 = \text{se}(\theta_0)$ is the standard error computed at the null value. We reject $H_0$ if $|W| > z_{\alpha/2}$.
The power function $\beta(\theta_\star)$ at the true parameter value $\theta_\star$ is the probability of rejecting the null hypothesis:
$$\beta(\theta_\star) = P_{\theta_\star} \left( \left| \frac{\hat{\theta} - \theta_0}{\text{se}_0} \right| > z_{\alpha/2} \right).$$
This inequality expands to two cases:
$$\frac{\hat{\theta} - \theta_0}{\text{se}_0} > z_{\alpha/2} \quad \text{or} \quad \frac{\hat{\theta} - \theta_0}{\text{se}_0} < -z_{\alpha/2}.$$
To evaluate these probabilities under the true parameter $\theta_\star$, we must standardize $\hat{\theta}$ using the true mean $\theta_\star$ and the true standard error $\text{se}(\theta_\star)$. Note that $\hat{\theta} \approx N(\theta_\star, \text{se}(\theta_\star)^2)$. Let $Z = \frac{\hat{\theta} - \theta_\star}{\text{se}(\theta_\star)} \sim N(0, 1)$.
We rearrange the rejection inequalities to isolate $Z$:

**1. Right Tail:**
$$\frac{\hat{\theta} - \theta_0}{\text{se}_0} > z_{\alpha/2}$$
$$\hat{\theta} > \theta_0 + z_{\alpha/2} \text{se}_0$$
$$\frac{\hat{\theta} - \theta_\star}{\text{se}(\theta_\star)} > \frac{\theta_0 - \theta_\star + z_{\alpha/2} \text{se}_0}{\text{se}(\theta_\star)}$$
$$Z > \frac{\theta_0 - \theta_\star}{\text{se}(\theta_\star)} + z_{\alpha/2} \frac{\text{se}_0}{\text{se}(\theta_\star)}.$$

**2. Left Tail:**
$$\frac{\hat{\theta} - \theta_0}{\text{se}_0} < -z_{\alpha/2}$$
$$\hat{\theta} < \theta_0 - z_{\alpha/2} \text{se}_0$$
$$\frac{\hat{\theta} - \theta_\star}{\text{se}(\theta_\star)} < \frac{\theta_0 - \theta_\star - z_{\alpha/2} \text{se}_0}{\text{se}(\theta_\star)}$$
$$Z < \frac{\theta_0 - \theta_\star}{\text{se}(\theta_\star)} - z_{\alpha/2} \frac{\text{se}_0}{\text{se}(\theta_\star)}.$$

The power function is approximately:
$$\beta(\theta_\star) \approx 1 - \Phi \left( \frac{\theta_0 - \theta_\star}{\text{se}(\theta_\star)} + z_{\alpha/2} \frac{\text{se}_0}{\text{se}(\theta_\star)} \right) + \Phi \left( \frac{\theta_0 - \theta_\star}{\text{se}(\theta_\star)} - z_{\alpha/2} \frac{\text{se}_0}{\text{se}(\theta_\star)} \right).$$
Comparing this to the result in Part (a) (where the denominator was $\widehat{\text{se}} \approx \text{se}(\theta_\star)$):
i. The critical value term $z_{\alpha/2}$ is now scaled by the ratio of the standard errors $\frac{\text{se}_0}{\text{se}(\theta_\star)}$.
ii. If the standard error is constant (does not depend on $\theta$), then $\text{se}_0 = \text{se}(\theta_\star)$ and the power functions are identical. However, if the standard error depends on $\theta$ (e.g., in Bernoulli or Poisson distributions), the effective critical value shifts, altering the power.

2. **[Asymptotic distribution of the LRT—simple $H_0$]**
Do Problem 10.33 in Casella-Berger page 513
(note: this problem is only asking you to complete the last step in the proof; you can find the whole Section 10.5 on Asymptotic Evaluations on Canvas)
**Solution:** From the proof of Theorem 10.3.1, using a Taylor series expansion, we have established that:
$$-2 \log \lambda(\mathbf{X}) \approx (\hat{\theta} - \theta_0)^2 [-l''(\hat{\theta}|\mathbf{x})].$$
We define the random variable of interest $Z_n$ as the signed square root of this quantity:
$$Z_n = (\hat{\theta} - \theta_0) \sqrt{-l''(\hat{\theta}|\mathbf{x})}.$$
To apply the asymptotic theorems, we manipulate this expression by multiplying and dividing by $\sqrt{n}$:
$$Z_n = [\sqrt{n}(\hat{\theta} - \theta_0)] \left[ \sqrt{-\frac{1}{n} l''(\hat{\theta}|\mathbf{x})} \right].$$
We analyze the convergence of the two bracketed terms separately.
(a) **First Term:** From **Theorem 10.1.12** (Asymptotic Efficiency of MLEs), we know that:
$$\sqrt{n}(\hat{\theta} - \theta_0) \xrightarrow{d} N(0, v(\theta_0)) = N \left( 0, \frac{1}{I(\theta_0)} \right),$$
where $I(\theta_0)$ is the expected Fisher information contained in one observation.
(b) **Second Term:** By the Weak Law of Large Numbers (WLLN), for the true parameter $\theta_0$:
$$-\frac{1}{n} l''(\theta_0|\mathbf{x}) = \frac{1}{n} \sum_{i=1}^n -\frac{\partial^2}{\partial \theta^2} \log f(X_i|\theta_0) \xrightarrow{p} E_{\theta_0} \left[ -\frac{\partial^2}{\partial \theta^2} \log f(X|\theta_0) \right] = I(\theta_0).$$
Since the MLE $\hat{\theta}$ is consistent ($\hat{\theta} \xrightarrow{p} \theta_0$) and the second derivative of the log-likelihood is continuous (by regularity conditions), it follows that the observed information at the MLE converges to the expected information:
$$-\frac{1}{n} l''(\hat{\theta}|\mathbf{x}) \xrightarrow{p} I(\theta_0).$$

Therefore, taking the square root:
$$\sqrt{-\frac{1}{n} l''(\hat{\theta}|\mathbf{x})} \xrightarrow{p} \sqrt{I(\theta_0)}.$$
Now we apply **Slutsky’s Theorem (Theorem 5.5.17)**, which states that if $X_n \xrightarrow{d} X$ and $Y_n \xrightarrow{p} c$, then $X_n Y_n \xrightarrow{d} cX$.
Let $X_n = \sqrt{n}(\hat{\theta} - \theta_0)$ and $Y_n = \sqrt{-\frac{1}{n} l''(\hat{\theta}|\mathbf{x})}$. Then:
$$Z_n = X_n Y_n \xrightarrow{d} \sqrt{I(\theta_0)} \times N \left( 0, \frac{1}{I(\theta_0)} \right).$$
Multiplying a Normal random variable by a constant $c$ scales the variance by $c^2$:
$$Z_n \xrightarrow{d} N \left( 0, I(\theta_0) \cdot \frac{1}{I(\theta_0)} \right) = N(0, 1).$$
Thus, we have shown that:
$$(\hat{\theta} - \theta_0) \sqrt{-l''(\hat{\theta}|\mathbf{x})} \xrightarrow{d} N(0, 1).$$
Finally, using the Continuous Mapping Theorem, the square of a standard normal random variable is a chi-squared random variable with 1 degree of freedom. Therefore:
$$-2 \log \lambda(\mathbf{X}) \approx Z_n^2 \xrightarrow{d} \chi^2_1.$$

3. **[Equivalence between hypothesis tests and confidence sets]**
Prove Theorem 10.10 in Wasserman page 155.
**Solution:** By definition, the Wald test rejects the null hypothesis $H_0$ at significance level $\alpha$ if and only if the absolute value of the test statistic exceeds the critical value:
$$\left| \frac{\hat{\theta} - \theta_0}{\widehat{\text{se}}} \right| > z_{\alpha/2}.$$
We proceed by manipulating this inequality algebraically to isolate $\theta_0$:
$$|\hat{\theta} - \theta_0| > z_{\alpha/2} \widehat{\text{se}} \iff \theta_0 < \hat{\theta} - z_{\alpha/2} \widehat{\text{se}} \quad \text{or} \quad \theta_0 > \hat{\theta} + z_{\alpha/2} \widehat{\text{se}}.$$
The confidence interval $C$ is defined as the open interval $(\hat{\theta} - \widehat{\text{se}} z_{\alpha/2}, \hat{\theta} + \widehat{\text{se}} z_{\alpha/2})$.
The condition derived above states that $\theta_0$ is strictly less than the lower bound of $C$ or strictly greater than the upper bound of $C$. This is logically equivalent to stating that $\theta_0$ is not contained within the interval $C$.
Thus, we reject $H_0 \iff \theta_0 \notin C$.

$$L(\theta|\mathbf{x}) = \prod_{i=1}^n f(x_i|\theta) = p_1^{y_1} p_2^{y_2} p_3^{y_3} p_4^{y_4} p_5^{y_5},$$
where $y_j = \text{number of } x_1, \dots, x_n \text{ equal to } j$. Consider testing
$H_0 : p_1 = p_2 = p_3 \text{ and } p_4 = p_5 \quad \text{versus} \quad H_1 : H_0 \text{ is not true.}$

**Solution:** We write $\nu$ as the degrees of freedom. Recall $\nu$ is the difference in the number of free parameters under $\Theta_1$ versus $\Theta_0$ (i.e., the parameter space for the alternative versus the null).
(a): $\nu = 1 - 0 = 1$.
(b): $\nu = 2 - 1 = 1$.
(c): $\nu = d - 0 = d$.
(d): $\nu = (k - 1) - 0 = k - 1$. Note that under the alternative, specifying the values of $k - 1$ coordinates of $p$ means we have fully specified $p$ since $\sum_i p_i = 1$.
(e): $\nu = 1$ by applying (f) with $I = 2$ and $J = 2$.
(f): $\nu = (I - 1)(J - 1)$. Explanation: Remember that the null is $H_0 : p_{ij} = p_i p_j$ for all $i \in \{1, 2, \dots, I\}$ and $j \in \{1, \dots, J\}$. Under the null, there are $(I - 1) + (J - 1)$ degrees of freedom. To see this, note one just has to specify $p_i$ and $p_j$ to completely specify all the $p_{ij}$ values (and fill out the contingency table and its row-wise and column-wise sums), and by the same reasoning in (d), it takes $I - 1$ and $J - 1$ values to do so.
The alternative turns out to have $IJ - 1$ free parameters. To see this, first note that for any given row $i$, there are $J + 1$ parameters $p_{i1}, \dots, p_{iJ}$ and $p_i$ subject to the constraint that $\sum_{j=1}^J p_{ij} = p_i$. Thus, there are $J$ free parameters. This is the case for every one of the $I$ rows inside the table, giving $IJ$ total free parameters. The $(I + 1)$th row is excluded from this calculation because if we have the first $I$ rows specified, we can infer the column sums. Finally, note that there is a constraint $\sum_{i=1}^I p_i = 1$, meaning we can infer $p_I$ from $p_1, \dots, p_{I-1}$. So we obtain $IJ - 1$.
Thus, $\nu = IJ - 1 - (I - 1) - (J - 1) = (I - 1)(J - 1)$.
(g): $\nu = 4 - 1 = 3$. The reason the null only has one free parameter is that if you specify $p_1$, then you can infer $p_2, p_3$. You then have $\sum_{i=1}^5 p_i = 3p_1 + p_4 + p_5 = 1$, meaning we can then infer $p_4 + p_5$. But $p_4 = p_5$, so we recover $p_4$ and $p_5$ by taking $(p_4 + p_5)/2$.

5. **[LRT]** Suppose that $X_1, \dots, X_n \sim N(\theta, 1)$. Describe the size $\alpha$ likelihood ratio test for testing the simple hypotheses: $H_0 : \theta = \theta_1$ versus $H_1 : \theta = \theta_2$.
**Solution:**
First, compute the likelihood ratio:
$$\Lambda(X_1, \dots, X_n) = \frac{L(\theta_1)}{L(\theta_2)}$$
$$= \frac{\exp(-\sum_{i=1}^n (X_i - \theta_1)^2/2)}{\exp(-\sum_{i=1}^n (X_i - \theta_2)^2/2)}$$
$$= \exp \left( \frac{1}{2} \left( \sum_{i=1}^n (X_i - \theta_2)^2 - \sum_{i=1}^n (X_i - \theta_1)^2 \right) \right)$$
$$= \exp \left( \frac{1}{2} (n(\theta_2^2 - \theta_1^2) - 2n\bar{X}_n(\theta_2 - \theta_1)) \right)$$
Note that we want to reject the null if the ratio above is very small, meaning the likelihood at the alternative (denominator) is large compared to the likelihood at the null (numerator).
Now consider two cases. When $\theta_2 < \theta_1$, $\Lambda(X_1, \dots, X_n)$ is an increasing function of $\bar{X}_n$. Thus, the rejection region is
$$\Lambda(X_1, \dots, X_n) \leq k \iff \bar{X}_n \leq k^*$$
for some $k^*$. Since $\bar{X}_n \sim N(\theta_1, 1/n)$ under the null hypothesis, we need
$$P(\bar{X}_n \leq k^* | H_0) = P_{\theta_1} \left( \frac{\bar{X}_n - \theta_1}{1/\sqrt{n}} \leq \frac{k^* - \theta_1}{1/\sqrt{n}} \right) = P \left( Z \leq \frac{k^* - \theta_1}{1/\sqrt{n}} \right) = \alpha,$$
where $Z \sim N(0, 1)$. Since $z_\gamma$ is the number such that $P(Z > z_\gamma) = \gamma$, we set
$$\frac{k^* - \theta_1}{1/\sqrt{n}} = z_{1-\alpha} \iff k^* = \theta_1 + z_{1-\alpha} \frac{1}{\sqrt{n}}.$$
Similarly, when $\theta_2 > \theta_1$, $\Lambda(X_1, \dots, X_n)$ is a decreasing function of $\bar{X}_n$, so it is small when $\bar{X}_n$ is large. Then, the rejection region is
$$\Lambda(X_1, \dots, X_n) \leq k \iff \bar{X}_n \geq k'.$$
Thus we need
$$P(\bar{X}_n > k' | H_0) = P_{\theta_1} \left( \frac{\bar{X}_n - \theta_1}{1/\sqrt{n}} > \frac{k' - \theta_1}{1/\sqrt{n}} \right) = P \left( Z > \frac{k' - \theta_1}{1/\sqrt{n}} \right) = \alpha,$$
where $Z \sim N(0, 1)$. Hence we set
$$\frac{k' - \theta_1}{1/\sqrt{n}} = z_\alpha \iff k' = \theta_1 + z_\alpha \frac{1}{\sqrt{n}}.$$
To summarize, when $\theta_1 < \theta_2$ we reject when $\bar{X}_n \leq \theta_1 + z_{1-\alpha}/\sqrt{n}$, and when $\theta_2 < \theta_1$, we reject when $\bar{X}_n \geq \theta_1 + z_\alpha/\sqrt{n}$.

6. **[More LRT]** Suppose that, $X_1, \dots, X_n$ is drawn from a Pareto distribution with density:
$$p(x|\theta, \nu) = \frac{\theta \nu^\theta}{x^{\theta+1}} \mathbb{I}_{[\nu, \infty)}(x), \quad \theta > 0, \nu > 0.$$
Construct the (generalized) LRT statistic for distinguishing the hypotheses:
$H_0 : \theta = 1, \nu \text{ unknown,}$
$H_1 : \theta \neq 1, \nu \text{ unknown.}$
Suppose Wilks’ approximation was accurate. Derive the critical region for the GLRT.
**Solution:** The log likelihood ratio is.
$$\log L(\theta, \nu) = \left( n \log \theta + n\theta \log \nu - (\theta + 1) \sum_{i=1}^n \log X_i \right) \mathbb{I}_{(-\infty, X_{(1)}]}(\nu).$$
Since $\log(\cdot)$ is monotone increasing function,
$$\hat{\nu}_{MLE} = X_{(1)},$$
and the first order optimality condition implies
$$\frac{\partial \log L}{\partial \theta} \bigg|_{\theta=\hat{\theta}, \nu=\hat{\nu}} = \left( \frac{n}{\hat{\theta}} + n \log X_{(1)} - \sum_{i=1}^n \log X_i \right) = 0 \implies \hat{\theta}_{MLE} = \frac{n}{\sum_{i=1}^n \log X_i - n \log X_{(1)}}$$
From Wilks’ theorem,
$$\log \Lambda(X) = \log \sup_{\nu \in \mathbb{R}_+, \theta=1} L(\theta, \nu) - \log \sup_{\nu \in \mathbb{R}_+, \theta \in \mathbb{R}_+} L(\theta, \nu)$$
$$= \log L(1, X_{(1)}) - \log L(\hat{\theta}_{MLE}, X_{(1)})$$
$$-2 \log \Lambda(x) = 2 \log L(\hat{\theta}_{MLE}, X_{(1)}) - 2 \log L(1, X_{(1)}) \rightsquigarrow \chi^2_1$$
$$\implies RR = \{X : -2 \log \Lambda(X) > \chi^2_{1,\alpha}\}.$$

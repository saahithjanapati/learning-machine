## How to use this series

Read in chronological order. Each file is written as a guided lesson (motivation -> walkthrough -> checkpoints -> pitfalls -> recap), so you can follow it like a mini textbook instead of standalone notes.

If you are short on time, prioritize:
1. `Jan29` (GD),
2. `Feb10` (subgradient method),
3. `Feb12` (projected subgradient),
4. `Feb17` (optimality),
5. `Feb19` (SGD).
